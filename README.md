## 3. 프로젝트 결과

- **+, -, x, ÷, %(나머지), C, = 기능은 정상 작동함**

- **x², ²√x, 1/x 기능은 오작동**
    
    ex) 8→ x²→ 8→ x² 이렇게 두 번 클릭해야 값이 나옴
    

- **CE, Back, +/-, . 기능은 구현 못함**
